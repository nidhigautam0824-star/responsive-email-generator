# Multi-Platform Email & Invoice Engine 📧

A specialized tool to generate mobile-responsive HTML invoices and OTP emails using table-based layouts for cross-platform compatibility (Gmail, Outlook).

## 🚀 Features
- **Dynamic Content:** Generates professional-looking emails based on user input.
- **Email-Safe Coding:** Uses table-based layouts instead of Flexbox/Grid to ensure the design doesn't break in Outlook or older Gmail versions.
- **Mobile Responsive:** Ensures the email content looks good on all screen sizes.

## 🛠️ Tech Stack
- HTML5
- CSS (Inline)
- JavaScript

## 💡 Why this is unique?
In the world of modern CSS, handling legacy email rendering engines is a specific skill. This project demonstrates my ability to work within strict constraints to achieve cross-platform consistency.
